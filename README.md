# README #

### What is this repository for? ###

Traits3D is a successor of my 3D function plotting library [QwtPlot3D](http://qwtplot3d.sourceforge.net).
It has been rewritten using recent C++ and OpenGL features.

Operating Systems:

* Windows
* Linux
* Android
